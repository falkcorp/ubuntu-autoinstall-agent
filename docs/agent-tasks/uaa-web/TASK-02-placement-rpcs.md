## Step-by-step

1. Run the ⛔ START HERE block, then every anchor grep above. Any zero-hit grep (at old AND mapped path) → STOP and report.

2. **Placeholder gate** — in `placement.rs`:
   ```rust
   pub const PLACEHOLDER: &str = "REPLACE_AT_PLACE_TIME";
   /// Keys whose values are secret slots. Matched case-insensitively as the
   /// final path segment of a YAML key (e.g. "luks_key:", "  password:").
   pub const SECRET_KEYS: [&str; 8] = ["luks_key", "luks_passphrase", "password",
       "passwd", "hashed_passwd", "wifi_psk", "token", "private_key"];
   /// Fail-closed scan of seed content. Err(reason) if ANY of:
   ///  - a SECRET_KEYS line whose value is non-empty and != PLACEHOLDER
   ///  - any line containing "BEGIN" + "PRIVATE KEY" (PEM key material)
   /// A SECRET_KEYS line whose value IS the literal PLACEHOLDER passes —
   /// placeholders are the POINT (secrets are injected at place time by the
   /// operator tooling, never carried over this RPC).
   pub fn check_placeholders_only(content: &str) -> Result<(), String>;
   ```
   Parse line-by-line: strip comments after `#` outside quotes is NOT required — keep it simple and strict: `let (k, v) = line.split_once(':')`, trim both, compare `k`'s last `.`/`_`-normalized segment against `SECRET_KEYS` case-insensitively, and `v` (with surrounding quotes stripped) against `PLACEHOLDER`. Strictness over cleverness: a false REJECT is acceptable (operator fixes the seed); a false ACCEPT is not.

3. **Atomic write helper** — `fn write_atomic(final_path: &Path, bytes: &[u8]) -> io::Result<()>`: `create_dir_all(parent)`, write `<final>.tmp.<pid>`, `fs::rename`. Every placement write goes through it; no direct `fs::write` to a final path anywhere in this file.

4. **`place_seed`** — validate hexmac/filename components (reject `/`, `..`, NUL → `invalid_argument`); run `check_placeholders_only` on every file's content (→ `invalid_argument` with the reason, nothing written); then `write_atomic` each seed file under `<webroot>/cloud-init/<hexmac>/`. Response `Ack`.

5. **`place_ipxe`** — same component validation; `write_atomic` to `<webroot>/ipxe/mac-<hexmac>.ipxe`. iPXE content is non-secret boot script; still run `check_placeholders_only` (defense in depth — same gate, same fail-closed).

6. **`flip_boot_target`** — resolve the host's iPXE file (per the request: hostname → file mapping mirrors the Python `find_ipxe_file_by_hostname`; the request message carries what the proto defines — re-read `web.proto`). Missing file → `Ok(response { ok: false, message: format!("No iPXE file found for {hostname}") })`. Present: read, `Regex::new(r"set menu-default \S+")` → `replace_all` with `set menu-default {target}` (ALL occurrences — Python `re.sub` parity), `write_atomic` back, return `ok: true, message: format!("Flipped {hostname} to {target}")`. Validate `target` against the four Decision-13 values (`local-disk|custom-autoinstall|pxe-disabled|pxe-grub`) plus the literal iPXE menu ids the Python accepts — reject unknown targets `invalid_argument` (re-verify the accepted set against `grep -n "boot-local-disk" scripts/autoinstall-agent.py`).

7. **`remove_host`** — delete the seed dir `<webroot>/cloud-init/<hexmac>/` and the iPXE file if present (`remove_dir_all` / `remove_file`, ignoring `NotFound`); always `Ok(Ack)` (idempotent). Same component validation first.

8. **Unit tests** (`#[cfg(test)] mod tests` in `placement.rs`, tempdir webroot):

   | Test | Asserts |
   |---|---|
   | `test_gate_rejects_real_secret` | `luks_key: hunter2-actual-secret` → `Err`; `password: "s3cret"` → `Err`; PEM `-----BEGIN OPENSSH PRIVATE KEY-----` → `Err` |
   | `test_gate_allows_placeholder_and_nonsecret` | **anti-over-suppression:** `luks_key: REPLACE_AT_PLACE_TIME` + ordinary fields (`hostname: len-serv-001`) → `Ok`, and a full `place_seed` with that content WRITES the file (a legitimate placeholder-bearing seed still places) |
